## WebRTC monitoring made easy

Peermetrics is a self-hosted WebRTC monitoring system that collects metrics from client devices, ingests them through a dedicated API service, and visualizes them in a rich dashboard — giving you the visibility you need to troubleshoot issues and deliver better real-time experiences.
